# victoria-store
un site des ventes en lignes

*1. Objectif du site*
Créer une plateforme simple, attrayante et fonctionnelle pour vendre des produits en ligne (vêtements, accessoires, tech, ou autres selon ton choix).

---

*2. Pages principales*
- *Accueil* (bannière, nouveautés, promos)
- *Produits* (par catégorie)
- *Détails produit* (photos, description, prix, bouton "Ajouter au panier")
- *Panier* (récapitulatif, total, bouton commander)
- *À propos*
- *Contact*
- *FAQ*

---

*3. Fonctions utiles*
- Recherche de produits
- Filtres (prix, taille, catégorie…)
- Panier dynamique (JS)
- Responsive design (mobile + desktop)
- Bouton de paiement (peut être fictif ou simulation)

---

*4. Techno recommandée*
- HTML/CSS
- JavaScript pour l’interaction
- Tu peux plus tard ajouter PHP + base de données si tu veux gérer les stocks/commandes.

Tu veux commencer aujourd’hui par la page *accueil* ou une autre ?
